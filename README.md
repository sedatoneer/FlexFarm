# FlexFarm - CSS Flexbox Learning Game

FlexFarm, CSS Flexbox mantığını oyunlaştırarak öğreten interaktif bir web uygulamasıdır. Çiftçi Celal'e yardım et, hayvanları kurtar ve Flexbox ustası ol!

![FlexFarm Screenshot](/public/social-preview.png)

## Özellikler

- **20 Bölümlük Müfredat:** `justify-content`, `align-items`, `flex-direction`, `order`, `flex-wrap` ve daha fazlası.
- **Hikaye Modu:** Çiftçi Celal ve hayvanların eğlenceli hikayesi.
- **İnteraktif Editör:** Canlı kod yazma ve anlık önizleme.
- **Responsive Tasarım:** Hem mobilde hem masaüstünde sorunsuz deneyim.
- **Yardımcı Araçlar:** Flexbox sözlüğü (Cheatsheet) ve ipuçları.
- **Görsel & İşitsel Geri Bildirim:** Başarı efektleri, sesler ve konfetiler.

## Teknolojiler

- **Core:** React, TypeScript, Vite
- **Styling:** Tailwind CSS
- **State:** React Hooks
- **Effects:** Canvas Confetti, Custom SVG Assets

## Kurulum
